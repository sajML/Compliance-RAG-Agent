# docs

Place demo assets here:

- **`demo.png`** — a screenshot of the running app (referenced as the hero image in the main `README.md`).
- **`demo.gif`** *(optional, even better)* — a short screen recording of a query running, so the pipeline animation is visible.

To use the GIF instead of the screenshot, change the `<img src="docs/demo.png" ...>` in the root `README.md` to `docs/demo.gif`.
